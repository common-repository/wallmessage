== Description ==

Wallmessage plugin for WordPress allows you to experience faster and better communication with your users, 
especially your customers in WooCommerce.

You can easily install this plugin and connect to the wallmessage api in the shortest time.

Through this service, you can automatically manage many processes of communication with your customers through 
WhatsApp and increase the trust and loyalty of your customers.

You can communicate with your customers from the moment the customer arrives and registers until the order is sent and finalized, 
and transfer a good experience to them.

This service is the best way to communicate and interact with your customers. 

Using this service, you can significantly reduce the cost of your SMS communication with your customers.

== Installation ==

Download the plugin from wordpress.org, open its package and upload the wallmessage folder via the plugin menu.
Activate the plugin through the Plugins menu in WordPress.
Go to the WordPress menu of your site and click on Wallmessage, in the Wallmessage Api Config section, 
get the api key and api address from https://app.wallmessage.com/ and enter it, 
and then save the settings. (Hooray you have successfully installed and activated the plugin)
